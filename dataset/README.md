# Diretório de Dataset (Ignorado pelo Git)

Esta pasta contém os dados brutos ou processados do projeto (Arquivos `.csv`).

Devido ao tamanho expressivo dos bancos de dados, os arquivos em extensão `.csv` desta pasta estão listados no `.gitignore` da raiz do projeto para evitar travamentos ou estourar o limite de *push* do Github (Acima de 100mb por commit).

**O que fazer:**
Se você for de um dos esquadrões (Alpha, Delta ou Omega) e acabou de dar um `git clone`, você precisará colar os arquivos CSV originais da Olist (disponíveis no Kaggle ou no seu PC local) manualmente aqui dentro desta pasta para rodar os scripts de Python.
