## Descrição

Com a confirmação do Henry sobre quais features mais causam atrasos, traduzir as descobertas técnicas em uma narrativa de negócio persuasiva focada em Supply Chain e Growth.

### Perguntas a Responder nos Slides

1. **Qual é o tamanho do problema?** Quantos pedidos atrasam por mês? Quanto isso custa em retenção?
2. **Quem são os vilões?** Quais rotas, categorias ou vendedores mais contribuem para os atrasos?
3. **Qual é a solução?** Como o modelo preditivo permite que a Olist aja ANTES do atraso acontecer?
4. **Qual é o impacto esperado?** Se a Olist usar o modelo para intervir em 50% dos atrasos previstos, quanto ela economiza em retenção?

### Fontes de Dados
- Resultados da EDA (ALPHA-03)
- Features selecionadas (ALPHA-06)
- Feature Importance (ALPHA-10, quando disponível)

## Critério de Aceite

- [ ] Narrativa respondendo as 4 perguntas acima
- [ ] Linguagem corporativa (sem jargão técnico de ML)
- [ ] Pelo menos 1 cálculo de impacto financeiro estimado
