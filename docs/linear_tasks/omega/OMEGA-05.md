## Descrição

Reunir todo o time para um ensaio completo da apresentação, testar o App ao vivo e corrigir os últimos bugs ou falhas de narrativa.

### Agenda do Ensaio

1. **Revisão dos Slides (30 min):**
   - Passar slide por slide verificando erros de português, dados e fluxo.
   - Definir quem apresenta cada bloco.

2. **Teste do App ao Vivo (20 min):**
   - Abrir o Streamlit (local ou Cloud) e navegar pelas 3 abas.
   - Testar o Simulador com inputs extremos (peso 0, frete 0, estado raro).
   - Verificar se o deploy Cloud está acessível publicamente.

3. **Ensaio Cronometrado (30 min):**
   - Apresentar como se fosse a banca real.
   - Cronometrar o tempo total.
   - Anotar pontos de melhoria.

4. **Correções Finais (tempo restante):**
   - Corrigir bugs encontrados no teste.
   - Ajustar slides conforme feedback do ensaio.
